Design a Creator Post Management System with Tags and Product Links

Problem Statement : Design a system for Wishlink that allows Creators to post content. Each Post can be tagged with multiple Tags and linked to one or more Products. Creators should be able to view their Posts, filter by Tags, and track which Products are associated with which Posts. Also, support the ability to retrieve trending Tags and popular Products based on engagement.

Requirements :
-Creator can post
-Post can have multiple tags, linked with multiple product
-Creator can view their product
-Filter by Tag
-Track post & Product Association
-Find trending Tags
-Popular product
